Class: Label

GraphPaper.Shapes. Label

Creates a new text box on the graph

new Label(point, text, align, centered, color, font)

Constructor for a new text field on a graph
Parameters:
Name Type Argument Default Description
point Point The point to put the text at
text string The text to display
align string <optional>
 "start" Text align
centered boolean <optional>
 false If true, it will attempt to put the point at the center of the text. Otherwise, it will put it where it would be for the given alignment
color string <optional>
 "black" The color of the text
font string <optional>
 "10px sans-serif" The font of the text. Default is the canvas default
