Class: Graph

GraphPaper. Graph

GraphPaper.Graph is the starting point of the library.

new Graph(canvas, xMin, xMax, yMin, yMax, axes, gridlines, tabs, style)

This is the main class GraphPaper.Graph. It is used to initialize a new graph. By default a graph initializes with a window of [-10, 10] by [-10, 10].
Parameters:
